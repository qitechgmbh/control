## 1. Specification, Features and Technical Data

### 1.1 Specification

- **Control Mode:** V/F Control  
- **Nominal Voltage:** 220 V  
- **Rated Power:** 750 W, 400 W, 200 W, 300 W  
- **Input Voltage:** Single Phase 220 V

- **Dimensions:** 52.00 / 80.00 / 90.00 / 147.00 mm (reference)

### 1.2 Features

- Output frequency range: **1.0–99.0 Hz**, suitable for general speed control
- Built-in intelligent logic controller for simple logic control
- Compact size, easy installation, simple commissioning
- Clear and easy-to-understand parameters for general applications
- Equipped with a built-in speed potentiometer; external potentiometer supported
- Specifically designed for **220 V, ≤0.75 kW three-phase AC motors**
- Integrated solution for equipment manufacturers, reducing system cost and improving reliability
- Electronic thermal overload protection, replacing traditional motor protection devices
- External LED indicator output supported for on-site indication
- User-friendly interface with simple parameter setting
- Adjustable V/F curve for special applications
- Shift key allows viewing real-time parameters
- Uses a new-generation **IPM module** with comprehensive protection functions

## 4. Parameters

### 4.1 Editing Parameters

1. **Enter Edit Mode**  
   - Press the **MENU/ESC** key.  
   - The digital display will flash `-0.1-`.  

2. **Select Function Code**  
   - Use the **↑/↓ keys** to adjust the value and choose the desired **function code**.  
   - Refer to **Table 4.1** below for available function codes.  

3. **Set Sub-Code**  
   - During the setting process, use the **P.K/SHIFT** and **↑/↓ keys** to select the sub-code.  
   - Press **ON/OFF** to enter the sub-code selection.  
   - After selecting the sub-code, press **ON/OFF** again to return to the function code screen (display flashes `-X.X-`).  
   - Repeat for the next function code as needed.

4. **Save Settings**  
   - After all settings are selected, press the **SAVE/LOCK** key (display flashes `SAVE`).  
   - Press **SAVE/LOCK** again to confirm saving.  
   - When the display stops flashing, the data is saved.  

5. **Operation After Saving**  
   - The VFD will run according to the saved parameters **without needing a power cycle**.  

6. **Exit Without Saving**  
   - Press **MENU/ESC** to exit without saving — previous parameters remain unchanged.  
   - If no key is pressed for **20 seconds**, the interface will automatically return to the run screen.

## 5. Fault Alarms and Countermeasures

The VFD has multiple warning indicators and protection functions.  
When an abnormal fault occurs, the protection function activates and the VFD stops output. The display panel shows a fault code.  

### Table 5.1 – Fault Codes

| Fault Code | Fault Name                         | Fault Description                                                                                  | Countermeasure                              |
| ------------| ------------------------------------| ----------------------------------------------------------------------------------------------------| ---------------------------------------------|
| E-0.1      | VFD Overheat                       | VFD temperature exceeds 1.6× set protection value; display flashes; max frequency limited to 30 Hz | After clearing the fault, restart the motor |
| E-0.2      | Pulse Overcurrent                  | VFD current exceeds 4.0× set protection value; motor stops                                         | After clearing the fault, restart the motor |
| E-0.4      | VFD Overload                       | VFD voltage × current exceeds 4.2× set power; display flashes; operation continues                 | After clearing the fault, restart the motor |
| E-0.5      | Approaching IGBT Temperature Limit | VFD temperature >120°C, near IGBT limit; motor stops                                               | After clearing the fault, restart the motor |
| E-0.6      | Temperature Sensor Open            | VFD temperature sensor open; display flashes; operation continues                                  | After clearing the fault, restart the motor |
| E-0.7      | Temperature Sensor Short           | VFD temperature sensor shorted; display flashes; operation continues                               | After clearing the fault, restart the motor |
| E-0.8      | 100% Overload                      | VFD overload warning lasts 6 s; motor stops                                                        | After clearing the fault, restart the motor |

## 7. Low-Frequency V/F Compensation

The VFD allows **low-frequency V/F compensation** to improve motor torque under load.  

- Based on the connected load, you can set values in parameters `-0.3-`, `-0.4-`, and `-0.5-` according to the **linear V/F curve** (see Table 7.1).  
- Low-frequency compensation **increases motor torque**.  
- You must choose the **upper frequency limit for torque boosting**:  
  - Set the **maximum voltage ratio** in `-0.3-` and `-0.4-` for the desired frequency.  
  - Reference the corresponding or nearest frequency in Table 2.  

**Behavior:**
- When operating **above this frequency**, the V/F slope increases → torque boost.  
- When operating **below this frequency**, the V/F slope decreases → torque decreases.  

**Example:**
- Parameter `-0.3-` = 20.0 Hz  
- Parameter `-0.4-` = 60, 50, 30  
- Parameter `-0.5-` default = 8  
- The three V/F curves correspond to different voltage ratios at 20 Hz:
  - Curve 1: 60% voltage ratio  
  - Curve 2: 55% voltage ratio  
  - Curve 3: 30% voltage ratio  

This allows fine-tuning of torque at low speeds while keeping proper V/F control at higher frequencies.
